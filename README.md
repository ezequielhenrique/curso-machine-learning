## 🚀 Projetos  

🔹 **Classificação - Aprovação de Empréstimos**  
- Objetivo: Prever a aprovação de empréstimos com base em dados históricos.  
- Modelos testados: Naive Bayes, SVM, Regressão Logística, KNN, Árvore de decisão, Random Forest e XGBoost.
